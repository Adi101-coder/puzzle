⏰ Clock Puzzle Quiz

📜 Rules & Instructions

  • The game consists of **6 clock images**, each representing a letter.
  • Analyze the **positions of the clock hands** carefully—they form a hidden word.
  • Type your guessed word in the input box below.
  • Click the **"Submit Answer"** button to check if your guess is correct.
  • ✅ If correct, you’ll see a **success message**!
  • ❌ If wrong, try again—you have **unlimited attempts**.
  • 🔎 **Hint:** Think about the relation between time and letters!
  • 🚀 **Challenge:** Solve it in the fewest attempts possible.
